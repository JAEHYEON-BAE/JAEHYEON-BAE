### 🔍 Lights — Stage Lighting Controller

> A full-featured live stage lighting application with custom Arduino firmware, cross-platform desktop app, and real-time DMX512 output.

**Background** &nbsp;|&nbsp; Needed a DAW-independent lighting controller for live performances that didn't require expensive proprietary hardware.

**What I built** &nbsp;|&nbsp; Electron + React desktop app (macOS & Windows) with scene management, cue lists, a real-time effect engine (Chase, Sine Pulse, Color Wave, Strobe), and custom Arduino Mega firmware handling the DMX512 serial protocol. Simulate mode allows full operation without physical hardware.

**Stack:** `Electron` `React` `JavaScript` `Arduino C` `DMX512`
&nbsp;&nbsp;[→ View Repository](https://github.com/JAEHYEON-BAE/Lights)
